Step-by-Step Bootloader Unlocking for Chinese Devices (For Users Outside China)

1. Creating a Chinese Xiaomi Account:

Clear your browser cookies.

Open the following link in your browser to register an account with China as the region:

https://cn.account.xiaomi.com/pass/register?sid=unlockApi&_locale=en_CN

Notes:

  • The following international phone numbers are not supported: +91, +62, +1
  • You can use free services available on Google to get a temporary phone number.
  • If you face issues receiving the verification code, try the following:
    • Open WhatsApp (if your phone number is registered there, the code might be sent to WhatsApp instead of SMS).
    • If the code still doesn’t arrive, add your region to the URL using: &uRegion=<your_region>
    • Example (for Russia): https://cn.account.xiaomi.com/pass/register?sid=unlockApi&_locale=en_CN&uRegion=RU
  • To find your region code, visit: https://account.xiaomi.com/pass2/config?key=areaConfig

2. Getting Permission from the Xiaomi Community App:

Note:

  • If you're using MIUI, skip this step (Step 2).
  • If you're using HyperOS and can bypass using MiBypassTool, skip both Step 2 and Step 3.

Log into the Mi Community Global app and request bootloader unlock permission.

If it's a new account, you must wait 30 days before this feature becomes available.
